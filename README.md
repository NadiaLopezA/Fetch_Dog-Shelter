## Getting Started

### Prerequisites

* Node.js:
  <a href="https://nodejs.org/en">Installation guide</a>

* npm
  ```sh
  npm install npm@latest -g
  ```

* yarn
   ```sh
   npm install --global yarn
   ```
  
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install dependencies
   ```sh
    yarn
   ```
3. Run the project locally
   ```sh
    yarn dev
   ```
